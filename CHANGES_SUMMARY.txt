FILE: frontend/src/components/GuidanceModal.jsx
═══════════════════════════════════════════════════════════════════════════════

CHANGES:

1. Added imports:
   - useRef: For panel positioning logic

2. Added state:
   - isExiting: Triggers exit animation on close

3. New function:
   - handleClose(): Initiates exit animation before calling onClose()

4. New effect (useEffect after line 105):
   - Smart position management
   - Detects collision with tour target
   - Moves panel up if would block interactive elements
   - Adjusts based on viewport height

5. Updated handlers:
   - Close button: Uses handleClose() instead of onClose()
   - Skip button: Uses handleClose()
   - Done button: Uses handleClose()

6. Updated render:
   - Added ref={panelRef} to panel element
   - Added className conditional: ${isExiting ? 'exiting' : ''}

RESULT: Modal now intelligently positions itself to avoid blocking chess board
and interactive elements. Smooth exit animation on close.
